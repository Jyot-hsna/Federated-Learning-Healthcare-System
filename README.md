# Federated Learning Healthcare System

This project implements a privacy-preserving federated learning framework 
for healthcare data using Python, PyTorch, and Flask.

## Features
- Decentralized model training
- ECC & Ring Signature security
- Differential Privacy implementation
- Flask-based monitoring dashboard

## Technologies Used
- Python
- PyTorch
- Flask
- SQLite
